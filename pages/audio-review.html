GSN Audio Reviewer

It is a full-stack application that allows users to log in to an account created for them by the Executive Producer of the project they worked on. The Producer can upload audio files for review, and all other users can review them.
The audio files are stored in an S3 storage container and the reviews are stored in a PostgreSQL database. The front-end is built with React and the back-end is built with Node and Express. The app is deployed on Heroku for now.

Technologies used:

  • Express

  • PostgreSQL

  • Amazon S3

Features:

  • Users can log in to an account created for them and review each and every project's audio file, leaving notes as they wish.

  • The Producer can upload audio files for review, (re)set usernames and passwords for their clients, and look at every user/project/note at the same time.
